当店について

菓匠 鈴乃家は、江戸時代から続く老舗和菓子店です。代々受け継がれてきた伝統の技と、新しい感性を融合させた和菓子づくりに取り組んでいます。

四季折々の素材を活かし、日本の美意識と文化を表現した和菓子を、心を込めてお作りしています。
